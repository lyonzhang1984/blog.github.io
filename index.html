某云平台CentOS7系统安装、配置、部署 Hexo静态博客

CentOS7系统安装和配置Hexo

一.安装Git

yum install git

git –version

配置邮箱和用户名：

git config --global user.email "you@example.com"
git config --global user.name "Your Name"

二.安装Node.js

通过nvm安装node
NVM地址：

脚本安装：

或者

如果无法下载nvm源码，可以采用手动安装：

  1. 下载安装包

  2. 添加环境变量（支持从镜像服务器下载数据包）

    添加到~/.bashrc

相关命令：

nvm install stable #安装Node最新稳定版

nvm list-remote #列出远程所有Node版本

nvm install v4.0.0 #安装v4.0.0

nvm ls #当前安装版本

nvm use v4.0.0 #切换版本

nvm alias default v4.0.0 #设置默认版本

nvm –version

node –version

三.安装Hexo

npm install -g hexo

mkdir /user/local/blog_hexo #本地安装路径

cd /usr/local/blog_hexo
hexo init #初始化

在_config.yml中进行基本配置:

hexo g
hexo s

可以访问如下地址进行验证:

http://localhost:4000

【注意】如果是某云环境，请将端口加入到安全端口列表，然后才可以提供互联网访问。

四.连接Github

新建repository

[your_user_name].github.io

[your_user_name] 必须是你的Github的名字，因为这个规则是有用的。

创建GitHub Pages

通过respository的Settings选项卡

https://[your_user_name].github.io

五.配置_config.yum

六.部署

//安装插件

npm install hexo-deployer-git --save

//部署

hexo d

七. 发布博客

hexo new ‘title’

hexo g

hexo d

Done. @20180601
